# Writer Vault Starter

这是 Offpedia P2 的示例 Obsidian Vault 模板。

## 目录

```txt
00 Inbox/       捕捉想法
10 Drafts/      草稿
20 Essays/      成稿
30 References/ 参考资料
40 Published/   已发布
80 Assets/      图片和附件
90 Templates/   模板
```

## 使用方式

1. 用 Obsidian 打开这个文件夹。
2. 从 `90 Templates` 复制文章模板。
3. 在 `10 Drafts` 开始写草稿。
4. 完成后移动到 `20 Essays`。
5. 发布后移动到 `40 Published`。
