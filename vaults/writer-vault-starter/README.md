# Writer Vault Starter

This is the example Obsidian vault template for Offpedia.

## Folders

```txt
00 Inbox/       Capture ideas
10 Drafts/      Drafts
20 Essays/      Finished essays
30 References/  Reference material
40 Published/   Published work
80 Assets/      Images and attachments
90 Templates/   Templates
```

## How to use

1. Open this folder in Obsidian.
2. Copy an essay template from `90 Templates`.
3. Start drafting in `10 Drafts`.
4. Move finished essays to `20 Essays`.
5. Move published work to `40 Published`.
