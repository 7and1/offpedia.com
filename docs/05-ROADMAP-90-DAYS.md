# 90-Day Roadmap

## Month 1

- Launch the GitHub Pages site.
- Ship pSEO v1 for compare, persona-goal, and workflow pages.
- Submit the sitemap to Search Console.
- Expand the tool library to 20-30 strong entries.
- Keep the page count controlled and monitor indexing quality.

## Month 2

- Add a static Stack Generator with a small Preact island.
- Inputs: persona, goal, offline needs, Git needs, publishing needs.
- Output: recommended stack, kit, guide, and next step.
- Add more kits only where a stack has real reuse value.

## Month 3

- Add contribution flow.
- Add contributor page.
- Add RSS.
- Add workflow health signals.
- Consider switching the repository public if community contribution becomes a priority.

## Success Signals

- Indexed pSEO pages
- Search impressions for long-tail queries
- Kit downloads
- Internal search usage
- Returning visitors on workflow pages
