# Product Requirements

## Product Name

Offpedia

## Positioning

Offpedia is a local-first workflow encyclopedia. It provides structured wiki entries, workflow stacks, downloadable kits, and practical setup guides.

## User Problem

Users do not lack tools. They lack working systems that remain portable, understandable, and maintainable over time.

Typical questions:

- How should I use Obsidian and GitHub for writing?
- How can notes stay local while still being synced and published?
- Should I choose Obsidian, Logseq, Notion, or another tool?
- Can I download a designed vault instead of starting from scratch?
- What are durable alternatives to cloud-locked document systems?

## User Value

A visitor should be able to:

1. Understand a tool or concept.
2. Choose a workflow stack.
3. Download a starter kit.
4. Follow a guide and complete the workflow.

## MVP Scope

- Home page
- Wiki entries
- Stack pages
- Kit pages
- Guide pages
- Compare pages
- Local search
- GitHub Pages deployment

## Non-goals

- Community forum
- SaaS editor
- Download mirror
- General encyclopedia
- Unstructured personal note archive

## Risks

- Content becomes scattered notes: use templates and frontmatter.
- The site becomes too Obsidian-specific: keep the framing around local-first workflows.
- Topics become too broad: focus on knowledge workflows first.
- Kits do not get used: place kit calls to action near stack recommendations.
