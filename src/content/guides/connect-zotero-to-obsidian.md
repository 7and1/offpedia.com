## What you will have

A repeatable handoff from citation library to literature notes, concept notes, and draft-ready project notes.

## Basic path

```txt
Zotero library -> literature note -> concept note -> project note -> draft
```

## Step overview

1. Keep Zotero as the source and citation library.
2. Decide how citation keys or exports will stay stable.
3. Create one literature-note template in Obsidian.
4. Split concept notes from source summaries.
5. Build project notes that only link to the evidence they actually use.

## Acceptance checks

- Can you trace one claim back to a Zotero item?
- Can you open a concept note without rereading the original PDF first?
- Can you see which literature notes feed one project note?
