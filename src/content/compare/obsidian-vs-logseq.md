## 结论

如果你主要写长文、做公开发布、维护 Markdown 文件夹，优先选 Obsidian。

如果你偏大纲式思考、块引用和每日笔记驱动，Logseq 很适合。

## 对比表

| 维度 | Obsidian | Logseq |
|---|---|---|
| 主要体验 | 文档 / 页面 | 大纲 / 块 |
| 数据格式 | Markdown 文件夹 | Markdown / Org 文件 |
| 长文写作 | 强 | 中 |
| 大纲思考 | 中 | 强 |
| 发布生态 | 强 | 中 |
| 新手门槛 | 中 | 中 |

## Offpedia 推荐

- 写作者：Obsidian + GitHub
- 研究者：Obsidian + Zotero 或 Logseq + Zotero
- 大纲思考者：Logseq
