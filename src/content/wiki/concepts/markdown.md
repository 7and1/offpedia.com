## 定义

Markdown 是一种用纯文本表达标题、列表、链接、引用、代码块等结构的轻量标记语言。

## 为什么 Offpedia 重视 Markdown

因为 Markdown 有三个优势：

1. 普通文本，长期可读。
2. 可被大量工具识别。
3. 适合 Git 版本管理。

## 常见误区

Markdown 不等于排版系统。它更像“内容结构格式”。如果需要出版级排版，可以和 Typst、Pandoc 或 LaTeX 搭配。
