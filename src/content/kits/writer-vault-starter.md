## Kit 简介

Writer Vault Starter 是给写作者准备的 Obsidian Vault 模板。

它不是一个复杂系统，而是一套能长期维护的最小结构：

- 想法捕捉
- 草稿管理
- 文章归档
- 参考资料
- 发布记录
- 模板复用

## 适合谁

- 写博客的人
- 写 newsletter 的人
- 写产品日志的人
- 需要长期管理草稿的人

## 文件夹结构

```txt
Writer Vault Starter/
├── 00 Inbox/
├── 10 Drafts/
├── 20 Essays/
├── 30 References/
├── 40 Published/
├── 80 Assets/
└── 90 Templates/
```

## 内置模板

- Essay Template
- Reference Note Template
- Daily Capture Template

## 安装方式

1. 下载 ZIP。
2. 解压到本地。
3. 用 Obsidian 打开这个文件夹。
4. 按 Guide 配置 GitHub 同步。
