## Kit overview

Writer Vault Starter is an Obsidian vault template for writers.

It is intentionally small. The structure covers:

- Idea capture
- Draft management
- Essay archiving
- Reference notes
- Publishing records
- Reusable templates

## Best for

- Blog writers
- Newsletter authors
- Product log writers
- People who need long-term draft management

## Folder structure

```txt
Writer Vault/
├── 00 Inbox/
├── 10 Drafts/
├── 20 Essays/
├── 30 References/
├── 40 Published/
├── 80 Assets/
└── 90 Templates/
```

## Included templates

- Essay template
- Reference note template
- Daily capture template

## Setup

1. Download the ZIP.
2. Unzip it locally.
3. Open the folder in Obsidian.
4. Follow the GitHub sync guide if you want version history and backup.
